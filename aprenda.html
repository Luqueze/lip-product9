Lógica de programação para iniciantes

Agora que você viu como fazer uma modificação em um jogo do scratch, é a hora de aprender o básico da lógica de programação

Programação pode parecer assustador, tantas letras, tantos números, tantos códigos diferentes podem levar pessoas a desistir de programar, porém eles são coisas que podem ser pesquisadas online que não há problemas

Porém logica de programação é algo que é nescessário saber para quem quer aprender a programar. Mas nem tudo que é nescessário é difícil como mostra a lógica de programação.

Tinkercad e Arduino

Vamos agora para algo mais interessante

Tinkercad foi feito para diversos usos. Aqui nós usaremos uma extensão que usa uma maquina chamada arduino.

Arduino é um mini computador que pode executar comandos simples. Ele é muito util para se começar a aprender programação bruta.

Tente começar sozinho. Pegue um projeto novo e tente fazer uma luz piscar a cada segundo usando blocos. Quando terminar passe o código para texto e tente entender o que acontece
